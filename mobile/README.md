# IMARAH — Aplikasi Mudah Alih (Native)

MVP web dan API dibina dahulu. Aplikasi mudah alih **native**:

- **iOS:** Swift (SwiftUI / UIKit)
- **Android:** Java (Android SDK)

Struktur projek Xcode / Android Studio akan ditambah pada fasa berikutnya. Gunakan API yang sama seperti portal web (`/api/...`).
